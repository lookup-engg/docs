### **1. How Updates Work**

Chrome automatically updates your extensions in the background, so in
most cases Velo Companion will stay up to date without any action from
you. However, if you want to force an immediate update - for example,
to get a new feature or bug fix right away - you can trigger one
manually.

### **2. Manually Updating the Extension**

To force an update:

- Type `chrome://extensions` in your browser's address bar and press
  Enter.

- Toggle **Developer Mode** on using the switch in the top-right
  corner of the page.

- Click the **Update** button that appears at the top of the page.

- Chrome will check for and apply updates to all installed extensions,
  including Velo Companion.

### **3. Confirming You Are on the Latest Version**

After updating, you can verify your version:

- On the `chrome://extensions` page, find **Velo Companion** in the
  list.

- The current installed version number is displayed below the
  extension name.

If you are unsure whether you have the latest version, you can compare
it against the version shown on the Chrome Web Store listing.
