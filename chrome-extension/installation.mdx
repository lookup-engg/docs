Install the chrome extension to capture Assisted Recording.

### **1. Install the Extension**  Go to the Chrome Web Store and search for **Velo**. Click Add to Chrome and then Add extension.  **2. Pin to Your Browser**

Make Velo easy to find by pinning it. Click the **puzzle icon** in your
browser bar, find Velo, and click the **pin icon**. This ensures you can
start a new project by just clicking at velo extension.

That's it, you are all set!
