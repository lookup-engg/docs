## Requirements

Before you start, make sure you have the following:

- **A Velo account** — sign in at [app.usevelo.ai](https://app.usevelo.ai)
- **The Velo Chrome Extension** — this method requires the extension to be installed and active. If you haven't installed it yet, see [Chrome Extension Installation](/chrome-extension/installation).
- **Your files ready** — Velo accepts PDF, PNG, JPG, GIF, and WebP files. You can upload up to **10 files** per Velo, with each file up to **100 MB**.

<Note>
  This method is currently in **Beta**. Some features may change as we improve the experience.
</Note>

## Supported file types

| Type | Extensions |
|------|------------|
| Documents | PDF |
| Images | PNG, JPG/JPEG, GIF, WebP |

For best results, use files that clearly show your product's UI — screenshots, slide decks exported as PDFs, or annotated images work well. Files with lots of text-heavy content or diagrams may produce less accurate results.

## Before you record

A quick narration pass goes a long way. Velo will open each file in a canvas session and you'll walk through them with your microphone on — talking through what the user is seeing. To make this smooth:

- Know roughly what you want to say for each file or page before you start
- Keep your narration focused on the product experience, not technical details
- Aim for 1–3 sentences per file or page
- Record somewhere quiet to keep the transcript clean

## Starting from the dashboard

From your Velo dashboard, click **Create New Velo**. On the creation method screen, click the **Upload PDFs & PNGs** card to begin.
