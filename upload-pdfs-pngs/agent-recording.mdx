---
title: "Agent Recording"
description: "Watch the AI agent record your demo in real time."
---

After you click **Go, Make my Velo →** on the Script Review page, Velo launches an AI agent to record your demo autonomously. You don't need to do anything — the agent takes over and works through your script scene by scene.

## What you'll see

The Agent Recording screen is split into two panels:

- **Agent Activity** (left panel) — a live log of what the agent is doing at each step, such as "Opening file…", "Navigating to scene 3…", or "Recording narration…". This updates in real time as the agent progresses through your demo.
- **Browser Agent** (right panel) — a live view of the agent's browser session. You can watch the agent interact with your files exactly as a real presenter would — moving through slides, focusing on UI elements, and progressing through each scene.

<Note>
  The agent session runs in an isolated browser environment managed by Velo. You do not need to interact with it.
</Note>

## Waiting for the agent

The agent works through the entire script automatically. Depending on the number of scenes and the target duration you set, this typically takes **1–3 minutes**.

You can watch the live feed or leave the page — Velo will process everything in the background and notify you when it's done.

## Cancelling the session

If you need to stop the agent mid-run, click **Cancel Session**. This will terminate the recording and return you to the Script Review page so you can make changes and try again.

<Warning>
  Cancelling the session will not consume your Velo Minutes for the cancelled run.
</Warning>

## After the agent finishes

Once the agent has completed all scenes, Velo processes the recording. You'll see a **Processing…** status while this completes.

When processing is done, you'll be taken to the **preview page** where you can watch your generated Velo and move into the editor to refine it.
