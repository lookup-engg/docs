Libraries are workspace-level collections of published videos. They are the primary way teammates discover and watch each other's work.

<Frame>
  ![Library view showing videos and sharing options](/images/workspaces/library-view.jpg)
</Frame>

## How libraries work

- Libraries are _flat_ — one level only. No sub-libraries or nesting.
- Every workspace starts with a default **General** library that all members can publish to immediately.
- All the members can create additional libraries regardless of roles.
- A single video can be published to _multiple_ libraries simultaneously.

## Public vs Private libraries

Each library has a visibility setting:

| Type | Who can see it | Who can add videos |
| --- | --- | --- |
| **Public** | All workspace members | Any workspace member |
| **Private** | Only invited members | Only invited members |

Private libraries are completely hidden from non-members — they don't appear in the sidebar or library list for anyone who isn't invited.

## Creating a library

**Step 1:** Click **Libraries** in the sidebar to expand the section.

**Step 2:** Click the **Create library** button.

**Step 3:** Name your library and choose its visibility (Public or Private).

## Publishing videos to a library

Publishing and sharing are two independent actions in Velo:

- **Publish** adds a video to one or more libraries so teammates can find it
- **Share** generates a public link for external viewers (see [Sharing](/creating-a-velo/sharing))

Neither requires the other. You can publish without sharing, or share without publishing.

To publish, select a video from **My Videos** or the chat panel, choose **Publish**, and select which libraries to add it to. Published videos remain in your **My Videos** as well.

## Browsing a library

Click a library name in the sidebar to view its contents. Videos are displayed as a grid showing the thumbnail, title, creator, duration, date, and view count.

Each video in a library has its own page with a video player and a **Docs** tab — an AI-generated written summary you can generate on demand.

## Managing libraries

From the library view, you can:

- Remove videos from the library
- Rename or delete the library (via the three-dot menu)
- Manage private library membership (Owner and Admin only)

<Note>
  Only Owners and Admins can create libraries and manage private library membership. All members can publish videos to any public library they have access to.
</Note>
